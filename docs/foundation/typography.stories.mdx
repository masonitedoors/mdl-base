## Headings

These are basics that are used for internal and external projects. But, the external usage of this is driven by Marketing. As an example, H1s on MCom are different in size than in internal products. They share the same font family, but the size usage could be different.

### Base Headings

<Canvas>
  <Story name="Base Headings">
    {`
      <h1>Header one</h1>
      <h2>Header two</h2>
      <h3>Header three</h3>
      <h4>Header four</h4>
      <h5>Header five</h5>
      <h6>Header six</h6>
    `}

### Heading Classes

.h1 through .h6 classes are also available, for when you want to match the font styling of a heading but cannot use the associated HTML element.

<Canvas>
  <Story name="Heading Classes">
    {`
      <p class="h1">Paragraph using .h1</p>
      <p class="h2">Paragraph using .h2</p>
      <p class="h3">Paragraph using .h3</p>
      <p class="h4">Paragraph using .h4</p>
      <p class="h5">Paragraph using .h5</p>
      <p class="h6">Paragraph using .h6</p>
    `}
  </Story>
</Canvas>

## Blockquotes

For quoting blocks of content from another source within your document. Wrap `<blockquote class="blockquote">` around any <abbr title="HyperText Markup Language">HTML</abbr> as the quote.

<Canvas>
  <Story name="Blockquote">
    {`
      <blockquote class="blockquote">
        <p>Show mangled quartz flip vibe exactly</p>
      </blockquote>
    `}
  </Story>
</Canvas>

### Naming a source

Add a `<footer class="blockquote-footer">` for identifying the source. Wrap the name of the source work in `<cite>`.

<Canvas>
  <Story name="Naming a source">
    {`
      <blockquote class="blockquote">
        <p>Show mangled quartz flip vibe exactly</p>
        <footer class="blockquote-footer">Someone famous in <cite title="Source Title">Source Title</cite></footer>
      </blockquote>
    `}
  </Story>
</Canvas>
